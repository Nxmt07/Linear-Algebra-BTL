# Linear-Algebra-BTL

A Python-based project that requires audio input for processing. Follow the steps below to get started.

## Prerequisites

- Python 3.x
- pip

## Installation

1. **Clone the repository**

   ```bash
   git clone <repository-url>
   cd Linear-Algebra-BTL
   ```

2. **Install dependencies**

   ```bash
   pip install -r requirements.txt
   ```

3. **Add your audio file**

   Place your audio file in the **same folder** as the cloned repository and rename it to:

   ```
   input.mp3
   ```

   Your folder structure should look like this:

   ```
   Linear-Algebra-BTL/
   ├── main.py
   ├── requirements.txt
   ├── input.mp3        ← your audio file goes here
   └── ...
   ```

## Usage

Run the main script:

```bash
python main.py
```

## Notes

- The audio file **must** be named exactly `input.mp3` and placed in the root of the project folder.
- Make sure all dependencies are installed before running `main.py`.
